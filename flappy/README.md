# Coin Runner

A fun endless runner game built with Flutter.

## About the Game

Coin Runner is an arcade-style game where players guide a character through a series of challenging obstacles. The objective is to survive as long as possible while collecting coins and achieving the highest score.

## Features

* Endless gameplay
* Coin collection system
* Score tracking
* Increasing difficulty
* Smooth Flutter animations
* Simple and responsive controls

## How to Play

1. Tap the screen to jump.
2. Avoid barriers and obstacles.
3. Successfully pass barriers to earn coins.
4. Collect additional coins along the way.
5. Try to beat your highest score.

## Coin System

* Passing each barrier rewards the player with coins.
* Coins are added to the total score.
* The longer you survive, the more coins you can earn.
